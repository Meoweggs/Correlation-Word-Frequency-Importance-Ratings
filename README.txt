This is the assignment for the "Methods of Social Survey and Social Research" course. I opened up the collected data and the code required to reproduce the results.
Wherein:
	'Questionnaire (in Chinese).docx' is a research questionnaire (sample service cost 315 RMB) in Chinese.
	'data0.sav' is the original data;
	'data1.csv' is the cleaned data. The data cleaning process includes removing invalid samples, redundant variables, setting grouping logic, and obtaining a total of 53 valid samples (N=53).
	'drawing.R' is the code used to reproduce the results (alas, my poor R), mainly for drawing.
